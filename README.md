# call--spam--detection-
📞 AI-Powered Scam Call Detection System
This project is an intelligent cybersecurity application designed to detect fraudulent scam calls using Natural Language Processing (NLP). The system analyzes call transcripts to classify them as either "Safe" or "Spam," providing an essential tool for personal security.

🚀 Key Features
Advanced NLP: Uses TfidfVectorizer for feature extraction and a trained classification model to identify scam patterns.

Modern UI: Built with Streamlit for a responsive, clean, and interactive user experience.

Cybersecurity Focus: Features a "Dark Mode" aesthetic designed for professional monitoring tools.

Authentication Flow: Includes a secure entry page to simulate a real-world enterprise application.

🛠 Tech Stack
Programming Language: Python 3.x

Machine Learning: scikit-learn (Naive Bayes, TF-IDF)

Web Framework: Streamlit

Version Control: Git & GitHub
